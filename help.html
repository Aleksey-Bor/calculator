Руководство пользователя

Приложение "Калькулятор" (далее калькулятор) предназначено для развлекательных целей. Приложение может работать только на компьютере с шириной экрана не менее 770px. Кнопки калькулятора нажимаются левой клавишей мышки.

Калькулатор позволяет совершать простые арифметические операции

  1. Cложение: +
  2. Вычитание: -
  3. Умножение: *
  4. Деление: /
  5. Возведение в степень: **
  6. Извлечение квадратного корня: √

Примеры последовательности действий необходимых для выполнения операций

  • Десять разделить на два добавить три: 10 ► / ► 2 ► + ► 3 ► = // 8
  • Ноль целых одна десятая умножить на ноль целых две десятые: 0 ► . ► 1 ► * ► 0 ► . ► 2 ► = // 0.02
  • Пять возвести во вторую степень: 5 ► ** ► 2 ► = // 25
  • Три возвести в третью степень: 3 ► ** ► 3 ► = // 27
  • Минус два возвести в третью степень: 2 ► +/- ► ** ► 3 ► = // -8
  • Квадратный корень из девяти: 9 ► V; // 3
  • Минус шесть плюс минус четыре: 6 ► +/- ► + ► 4 ► +/- ► = // -10
Клавиша С - очищает результат.
Клавиша СE - удаляет из памяти последнее введенное занчение.
